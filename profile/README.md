## Plugins

| Plugin                                                            | What it does                                                  |
| ----------------------------------------------------------------- | ------------------------------------------------------------- |
| [**cc-config**](https://github.com/clever-cc-plugins/cc-config)   | Bootstrap and audit a best-practice Claude Code configuration |
| [**cc-content**](https://github.com/clever-cc-plugins/cc-content) | A suite of content creation skills for marketing projects     |
| [**cc-chime**](https://github.com/clever-cc-plugins/cc-chime)     | Audio cues when Claude finishes a turn or needs your input    |

```
/plugin marketplace add clever-cc-plugins/marketplace
/plugin install cc-config@clever-cc-plugins
```

See the [marketplace](https://github.com/clever-cc-plugins/marketplace) repo for the full catalog, update instructions, and how to add your own plugin.

## Philosophy

> Every line costs tokens on every message.

These plugins are built around that one rule. No speculative config, no bundled dependencies, no skill that loads before it's needed. If a line isn't earning its place in your context window, it doesn't ship.
